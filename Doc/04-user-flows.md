# User Flows

## Seller Flow
1. Seller signs up
2. Mini-store is created
3. Seller adds products
4. Orders are received
5. Payouts processed by platform

## Customer Flow
1. Customer opens creator link
2. Browses products
3. Adds to cart & checks out
4. Receives order updates
5. Uses AI chat for support

## Order Flow
- ParentOrder created at platform level
- StoreOrder created for individual seller
- Payment → confirmation → payout
