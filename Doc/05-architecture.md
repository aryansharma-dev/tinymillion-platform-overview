# Architecture

TinyMillion uses a modern, scalable multi-tenant architecture.

## Frontend
- React + Tailwind
- Customer App
- Mini-Admin
- Global Admin

## Backend
- Node.js + Express
- REST APIs
- JWT authentication

## Database
- MongoDB
- Store-level data isolation

## Payments
- Razorpay
- Stripe
- Webhook-based confirmation

Designed for scalability and security.
